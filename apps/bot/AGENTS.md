# Bot (grammY)

- Бот **только принимает** файлы и открывает Mini App. Бизнес-логика — в API.
- Фото принимать как `document`, а не `photo`: Telegram сжимает `photo` до неузнаваемости.
- Bot API отдаёт `file_id`, а не файл. Скачивание — лимит 20 МБ.
- Все хендлеры идемпотентны: Telegram может доставить апдейт повторно.
- Long polling — только в dev. В проде — webhook.
